BONKCHAIN AKA VENOMCHAIN LEDGER v2.1 (REBRAND)
############################

BONKCHAIN is basic file based smart contract blockchain design to run from any web server.

BonkChain requires validators to lock crypto in order to recieve a copy of the sync keys.
Sync keys give access to the latest copy of the blockchain. Any out of sync validator will get penalize
Validators get rewarded with governance tokens per block.
